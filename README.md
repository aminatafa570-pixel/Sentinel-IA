# 🛡️ Sentinel IA

## AI-Powered Cybersecurity Platform

Sentinel IA is an intelligent cybersecurity platform designed to detect, analyze and respond to cyber threats in real time.

It combines artificial intelligence, risk analysis and human expertise to help security analysts make faster and more reliable decisions.

---

## Features

-  Real-time alerts
-  AI-powered threat analysis
-  Security dashboard
-  Audit logs
-  Incident management
-  AI recommendations
-  Live cyber attack map
-  Attack timeline
-  Risk scoring engine
-  Attack simulation
